Состав библиотеки:
	cbx2grd - отдельная кнопка для календаря (сначала хотел делать на основе грида потому и 2grd)
	topcld - Класс формы календаря 
	DateCnt - Контрол ввода дат (именно его надо бросать на форму :)

Быстрые клавиши
В контроле
	Shift+Enter - вызвать календарь
В форме
	Вправо/влево - День (-/+)
	Вверх/вниз - Неделя (-/+)
	PageUp/PageDn(Скрол мыши) - Месяц (-/+)
	Ctrl+PageUp/PageDw - Год (-/+)
	HOME - Сегодняшнее число
	END - Число по умолчанию
	Enter(Двойной щелчек) - Выбрать '+CHR(13)+;
	Esc - Отмена
